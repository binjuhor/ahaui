# Tag Input

## Introduction

Tag input component allows users enter one or several tags to an input field.

This component is based on [react-tagsinput][1]

  </li>
  <li>
    Tags
    <i className="u-text200"> - The main values of the field, with removable tags allow users to remove.</i>
  </li>
  <li>
    Button
    <i className="u-text200"> - A button is separated from the field as default. In some cases, the button can be put next to the field, on the right side, as the form group.</i>
  </li>
</ol>
