# Multi Steps

## Introduction

The component is used when a progress needs to be divided into multiple logical and numbered steps.

  src={require('src/assets/structure/stb_multi-steps-structure.jpg')}

  <li>

    <span className="u-marginLeftTiny u-text200">(optional)</span>

    <i className="u-text200"> - Display short & descriptive name/title of the step.</i>

      Progress bar
    <i className="u-text200"> - Indicate the current progress of the process.</i>

      Substrate

  </li>
</ol>
