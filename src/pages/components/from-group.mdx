# Form Group

## Introduction

Input group wraps inputs and buttons for more complex components.

## Examples

### Basic example

Place one add-on or button on either side of an input. You may also place one on both sides of an input. Remember to place  `<label>` above the input group.

<CodeView codeText={groupBasic} />
<CodeView codeText={group2Basic} />
<CodeView codeText={group3Basic} />

### Search box

A search box acts as the field for a query input or search term from the user to search and retrieve related information from the database. It typically appears as a single-line text box and is usually accompanied by a search button that initiates the search command.
