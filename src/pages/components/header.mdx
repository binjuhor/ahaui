# Header

## Introduction

Header structure is a template that explains the areas in the header and how it work which helps combine other components to create a consistent user experience.

### Default

<img
  src={require('src/assets/structure/default-header-structure.jpg')}
  srcSet={`${require('src/assets/structure/default-header-structure@2x.jpg')} 2x`}
  alt="Header structure"
  className="u-maxWidthFull"
/>
<ol className="u-textGray u-marginTopTiny">
  <li>Container: Using container to wrap header contents and grid system to set fix-width or full-width header.</li>
  <li>Logo: Using this area for logo of the site.</li>
  <li>Main content wrapper: Used to wrap all the header contents, except logo. This part is also replaced by a hamburger button on mobile.</li>
  <li>Right: Often used to present icons, CTA buttons or components about users' account.</li>
  <li>Left: Often used to display top menu of the site.</li>
</ol>

export const query = graphql`
  query HeaderQuery {
    Header: componentMetadata(displayName: { eq: "Header" }) {
      ...ComponentApi_metadata
    }
    HeaderContext: componentMetadata(displayName: { eq: "HeaderContext" }) {
      ...ComponentApi_metadata
    }
    HeaderMobile: componentMetadata(displayName: { eq: "HeaderMobile" }) {
      ...ComponentApi_metadata
    }
  }
`;
