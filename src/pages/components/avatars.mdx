# Avatars

## Introduction
An avatar is a circular component, which is used to represent people (askers & experts) and objects (system). Avatars can be photos, illustrations or icons.

## Examples

### Variants

#### Default avatar

When no image is available, a default silhouette is used.

Uploaded by users. When an image hasn't been uploaded, the default avatar will be used.
